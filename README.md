# DOB Calculator - Date of Birth Calculator

## 📅 About

The DOB Calculator is a fast, accurate, and user-friendly web tool designed to calculate age instantly from a given date of birth. Built with a modern Neumorphism (Soft UI) design, it offers a clean and intuitive experience across all devices.

This tool helps users determine their exact age in years, months, and days without manual calculations. It is perfect for personal use, official documentation, educational purposes, and eligibility checks.

---

## ✨ Key Features

- Accurate age calculation (years, months, days)
- Clean Neumorphic (Soft UI) interface
- Fully responsive and mobile-friendly
- Instant results with high performance
- No data storage or tracking
- Lightweight and easy to use

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Neumorphism Design)
- Vanilla JavaScript
